Learn HTML

HTML (HyperText Markup Language) is the standard language for creating webpages and web applications. It provides the structure of a page using elements and tags.

Basic Structure

Every HTML document starts with a <!DOCTYPE html> declaration, followed by <html>, <head>, and <body> elements.

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>
