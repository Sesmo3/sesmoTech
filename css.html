Learn CSS

CSS (Cascading Style Sheets) controls the style and layout of web pages. It describes how HTML elements should be displayed.

Basic Syntax

CSS rules consist of selectors and declaration blocks. The selector points to the HTML element to style, and declarations specify the property and value.

selector {
  property: value;
}

Example: Changing Text Color

p {
  color: red;
}

Example: Background Color

body {
  background-color: #f0f0f0;
}
